# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of
conduct. The full text is published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: be respectful and constructive, assume good faith, and help keep this a
welcoming community for everyone.

## Reporting

If you experience or witness unacceptable behavior, report it **privately** to
**conduct@ledticker.dev**. All reports are reviewed and handled confidentially,
and the maintainer is obligated to respect the privacy and security of the
reporter.
