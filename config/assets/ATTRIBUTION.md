# Sample-media attribution

The repo ships project-generated sample media — the Firebird phoenix family and a CC0
rainbow pride flag — used by the example configs and docs demos. The phoenix formats are
derived from a single CC0 source by `tools/derive_phoenix_assets.py`; the pride flag is
generated by `tools/derive_pride_assets.py` (see its entry below).

- **Source:** "pixel-phoenix" by **zonked** — https://opengameart.org/content/pixel-phoenix
- **License:** CC0 1.0 (public domain) — https://creativecommons.org/publicdomain/zero/1.0/
  No attribution is required; this record is kept for provenance.
- **Vendored source:** `config/assets/_src/phoenix-cc0-no-bg.gif` (20×20, animated, transparent).
- **Derived:** `phoenix.gif`, `phoenix_transparent.gif`, `phoenix.png`, `phoenix_transparent.png`,
  `phoenix.webp` (220×220, nearest-neighbor upscale).

## Pride flag demo assets

Both files are **project-generated CC0** — no third-party source or attribution required.
Generated by `tools/derive_pride_assets.py` (run `make derive-pride` to reproduce from scratch).

### `pride.gif` (1000×700) — 6-stripe rainbow flag

- **Design:** six solid horizontal bands — red `(228,3,3)`, orange `(255,140,0)`,
  yellow `(255,237,0)`, green `(0,128,38)`, blue `(0,77,255)`, violet `(117,7,135)` —
  scrolled vertically over 12 frames for a seamless animated loop.
- **License:** CC0 1.0 (public domain) — https://creativecommons.org/publicdomain/zero/1.0/

### `pride_trans.gif` (498×280) — 5-stripe trans pride flag

- **Design:** five symmetric horizontal bands — light-blue `(91,206,250)`, pink `(245,169,184)`,
  white `(255,255,255)`, pink `(245,169,184)`, light-blue `(91,206,250)` —
  scrolled vertically over 12 frames for a seamless animated loop.
- **License:** CC0 1.0 (public domain) — https://creativecommons.org/publicdomain/zero/1.0/

## Heart-tunnel backdrop

`heart-tunnel-opaque.jpg` (700×350 RGB JPEG) is **project-generated CC0** — no third-party
source or attribution required. Generated by `tools/derive_heart_tunnel.py` (run `make derive-heart-tunnel`
to reproduce from scratch).

- **Design:** concentric hearts receding to a center vanishing point; palette cycles
  trans-pride colors light-blue `(91,206,250)` → pink `(245,169,184)` → white `(255,255,255)` →
  pink (symmetric outward).
- **License:** CC0 1.0 (public domain) — https://creativecommons.org/publicdomain/zero/1.0/

Real-brand / customer-IP media is not committed — users add their own locally (drop files
into `config/assets/` and reference them in config).
