# Calendar Widget Options

Fetches upcoming events from any subscribed iCal (`.ics`) feed and displays them as a rotating agenda or a live next-event countdown. Always a Container: a background task polls the feed and populates `feed_stories`; the display loop re-reads the list on every pass, so updates appear within one cycle without restarting.

| Option                 | Type                                                                                  | Default                  | Description                                                                                                                                                                                                                                                                                       |
| ---------------------- | ------------------------------------------------------------------------------------- | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ics_url`              | string                                                                                | required                 | Public `.ics` URL to subscribe to (e.g. Google Calendar "Secret address in iCal format"), or a `file://` path for a local file (e.g. `"file:///home/pi/cal.ics"`).                                                                                                                                |
| `layout`               | `"agenda"` \| `"next"` \| `"two_row"`                                                 | `"agenda"`               | `"agenda"` shows one scrolling line per event; `"next"` shows a single live countdown to the soonest event (`"Standup in 25m"`); `"two_row"` shows one card per event â€” held day+time on top, scrolling title below.                                                                              |
| `max_events`           | int                                                                                   | `5`                      | Maximum number of upcoming events to display in agenda mode.                                                                                                                                                                                                                                      |
| `lookahead_days`       | int                                                                                   | `7`                      | Days ahead to scan for events. Recurrence rules are expanded within this window.                                                                                                                                                                                                                  |
| `time_format`          | `"12h"` \| `"24h"`                                                                    | `"12h"`                  | Format for the event time in agenda lines and the held top (when) row of `two_row` cards. `"12h"` renders `3:00 PM`; `"24h"` renders `15:00`.                                                                                                                                                     |
| `timezone`             | IANA name \| none                                                                     | system local             | Display timezone override, e.g. `"America/New_York"`. Uses stdlib `zoneinfo`. Defaults to the system local timezone.                                                                                                                                                                              |
| `empty_text`           | string                                                                                | `"No upcoming events"`   | Text shown when the feed loads but has no events in the lookahead window.                                                                                                                                                                                                                         |
| `error_text`           | string                                                                                | `"Calendar unavailable"` | Text shown when the feed fails to load on the first try (bad/unreachable `ics_url`). Kept distinct from `empty_text` so a broken feed doesn't look like an empty one; a later transient failure keeps the last-good events instead.                                                               |
| `filter`               | list of strings                                                                       | `[]` (all events)        | Keep only events whose summary contains any of these keywords (case-insensitive). An empty list shows all events.                                                                                                                                                                                 |
| `highlight`            | list of strings                                                                       | `[]` (none)              | Matching events are colored with `highlight_color` and guaranteed to appear even if they would otherwise be dropped by `max_events` capping.                                                                                                                                                      |
| `highlight_color`      | RGB list / string / table                                                             | amber `[255, 200, 60]`   | Color applied to highlighted events. Constant `[r,g,b]`, `"rainbow"`, `"color_cycle"`, `"random"`, or `{style="gradient", from=[...], to=[...]}`.                                                                                                                                                 |
| `font`                 | string                                                                                | `"6x12"`                 | BDF font name (e.g. `"5x8"`, `"6x12"`) or hires font (e.g. `"Inter-Bold"`). In `two_row` mode the default `6x12` is too tall for a split row, so it falls back to `5x8`; pick a fitting `font`/`font_size` for larger text (`led-ticker validate` rejects one too tall).                          |
| `font_color`           | RGB list / string / table                                                             | white `[255, 255, 255]`  | Color of the **event title** on non-highlighted lines. Constant `[r,g,b]`, `"rainbow"`, `"color_cycle"`, `"random"`, or `{style="gradient", from=[...], to=[...]}`.                                                                                                                               |
| `time_color`           | RGB list / string / table                                                             | amber `[255, 200, 60]`   | Color of the **time/relative phrase** on non-highlighted lines (e.g. `Tomorrow 3:00 PM Â·` in agenda, `Â· in 5m` in next). Same provider forms as `font_color`. Together these give the default two-tone look; set both equal for a single color.                                                   |
| `bg_color`             | RGB list                                                                              | none                     | Background fill color. Painted across the full panel before text.                                                                                                                                                                                                                                 |
| `border`               | `"rainbow"` \| `"color_cycle"` \| `"lightbulbs"` \| `[r,g,b]` \| `{style="...", ...}` | none                     | Perimeter border ring â€” five styles (rainbow chase, color cycle, constant, bands, lightbulbs); see [/concepts/borders/](/concepts/borders/).                                                                                                                                                      |
| `padding`              | int                                                                                   | `6`                      | Horizontal padding (logical pixels) added to each event line when scrolling.                                                                                                                                                                                                                      |
| `top_row_height`       | int                                                                                   | half the canvas          | **two_row only.** Logical rows for the held top (when) row; the bottom (title) row takes the rest, so it must be less than the section's `content_height`. Default splits the canvas 50/50. Raise it to give the day+time (top) row more room; lower it to give the title (bottom) row more room. |
| `top_text_y_offset`    | int                                                                                   | `0`                      | **two_row only.** Vertical nudge (logical pixels) for the top row's text.                                                                                                                                                                                                                         |
| `bottom_text_y_offset` | int                                                                                   | `0`                      | **two_row only.** Vertical nudge (logical pixels) for the bottom row's text.                                                                                                                                                                                                                      |
| `update_interval`      | int                                                                                   | `900`                    | Seconds between feed fetches. Default is 15 minutes.                                                                                                                                                                                                                                              |
