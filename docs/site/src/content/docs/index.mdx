## Writing a plugin

`led_ticker.plugin` is the public surface plugins import from — it exposes
widget, transition, emoji, font, and lifecycle-hook registration, no fork of
core needed. The pool water-temperature widget shown above is a real plugin
that lives in
[the led-ticker-plugins monorepo](https://github.com/JamesAwesome/led-ticker-plugins)
and contributes `type = "pool.monitor"`.

- [Plugins overview](/plugins/) — install, load, and the `[plugins]` config block
- [Write a plugin](/plugins/authoring/01-scaffold/) — scaffold, build, and package one
