Plugins live in their own repositories. To use one, add it to `config/requirements-plugins.txt` and rebuild — see [Plugins](/plugins/) for the install flow. Each plugin's own README documents its options, layouts, and setup.

## Widgets

- **[led-ticker-pool](https://github.com/JamesAwesome/led-ticker-pool)** — pool water-temperature widget (`type = "pool.monitor"`), backed by an InfluxDB v2 server. Cycles current / today / 7-day / season temperatures with zone coloring; supports `ticker` and `two_row` layouts. Full docs + screenshots in the repo README.
