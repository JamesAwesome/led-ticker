## moonbunny

|                     |                                                |
| ------------------- | ---------------------------------------------- |
| **Sign type**       | bigsign                                        |
| **Location**        | storefront window                              |
| **Hardware**        | Pi 5 + 8× P3 32×64                             |
| **Notable widgets** | `feeds.rss`, `two_row`, custom messages, `gif` |

moonbunny is an aerial-arts studio whose street-facing window runs
led-ticker as a 256×64 storefront display. The sign cycles brand info,
the studio's social handle on a held top row + scrolling promo on the
bottom (`two_row`), upcoming class blurbs as plain `message` widgets,
and short looping `gif` clips between sections.

The palette leans on the brand pink (`[225, 48, 108]`) for the social
handle and most headline text, with `font_color = "rainbow"` reserved
for occasional countdown widgets so a class going live still grabs
attention against the steadier brand-colored content. The bigsign
runs at `default_scale = 4` with a custom `pixel_mapper_config` for the
2×4 vertical-serpentine panel layout.

The display runs daily during business hours on a Raspberry Pi 5
behind the storefront, configured from a single `config.toml` and
restarted by systemd if the process exits. The same image runs in
dev on a Pi 4 + smallsign for layout iteration before promoting
changes to the storefront.
