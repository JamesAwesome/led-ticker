Sprite transitions run a pixel-art character across the panel, erasing the outgoing widget and revealing the incoming one. Reference them by their namespaced slug, e.g. `transition = "pokeball.forward"`.

## Homage transitions (flair pack)

The four homage families — **nyancat**, **pokeball**, **pacman**, and **sailor_moon** — ship together in the `led-ticker-flair` package (one install, four entry points, each with `.forward`, `.reverse`, and `.alternating` variants).

### Install

Add one line to `config/requirements-plugins.txt` (copied from `config/requirements-plugins.example.txt`):

```text
git+https://github.com/JamesAwesome/led-ticker-plugins.git@flair-v0.1.0#subdirectory=plugins/flair
```

One install registers all four families. See [Plugins](/plugins/) for the install flow.

A bare `transition = "nyancat"` (the pre-plugin spelling) raises a config-load error pointing at the namespaced `nyancat.forward` slug — the namespaced form is what the flair pack registers.

:::note[Existing configs need no change]
The type strings (`nyancat.forward`, `pokeball.alternating`, etc.) are unchanged — only the install source consolidates. Configs already using these transitions keep working as-is. If you previously pinned individual family tags (e.g. `@nyancat-v0.1.0`), those installs continue to work; switch to the single flair line at your convenience.
:::

### Nyan Cat (`nyancat`)

Nyan Cat flies across the panel with a rainbow trail. Auto-activates a hi-res webp sprite on scaled signs (see [Hires](#hires) below).

| Type string           | Direction            |
| --------------------- | -------------------- |
| `nyancat.forward`     | Left → right         |
| `nyancat.reverse`     | Right → left         |
| `nyancat.alternating` | Alternates each swap |

## Hires

`nyancat.forward` and `pokeball.forward` auto-activate hi-res sprites whenever the canvas is at `default_scale > 1` — that is, any big / scaled sign (see [Display](/concepts/display/)), typically a bigsign-style build. The hi-res variants are noticeably more detailed and read better at distance:

- **nyancat** — animated webp Nyan Cat sprite at native physical resolution
- **pokeball** — rolling pokeball with animated run-cycle sprite

The `sailor_moon` and `pacman` families keep their 8-bit look at all scales — the chunky aesthetic is part of the design.

## Other plugin sprites

The rolling-**baseball** family (`baseball.roll`, `baseball.roll_reverse`, `baseball.roll_alternating`) ships with the **[baseball](https://github.com/JamesAwesome/led-ticker-plugins/tree/main/plugins/baseball)** package in the led-ticker-plugins monorepo, alongside that package's MLB widgets and `:baseball.ball:` emoji. Install it via `config/requirements-plugins.txt` (see [Plugins](/plugins/)) to use it.
