The **calendar** widget is provided by the **[calendar](https://github.com/JamesAwesome/led-ticker-plugins/tree/main/plugins/calendar)** package in the led-ticker-plugins monorepo, referenced in config as `type = "calendar.events"`. It pulls upcoming events from any subscribed iCal (`.ics`) feed and shows them as a rotating **agenda**, a live **next**-event countdown, or a held-top **two_row** card.

- **To add it to your sign:** add `led-ticker-calendar` to `config/requirements-plugins.txt` and rebuild — see [Plugins](/plugins/) for the full flow.
- **Full documentation** — options, the `agenda` / `next` / `two_row` layouts, keyword filter/highlight, and colors — lives in the [calendar package README](https://github.com/JamesAwesome/led-ticker-plugins/blob/main/plugins/calendar/README.md).

See the [plugin system](/plugins/) for how plugin widgets work.
