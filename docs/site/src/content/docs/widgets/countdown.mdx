The `countdown` widget is a `message` with date arithmetic baked in. Every draw it
renders `<text>: <N>` where N is the number of days from today to `countdown_date`.
The count is recomputed on each draw, so it rolls to the next value at midnight
without restarting the process.

Once `countdown_date` has passed, the widget no longer displays — it drops out of
the rotation rather than showing a negative number. To count up from a past date
instead, see [countup](/widgets/countup/).

## Tips

### Typewriter is not available on `countdown`

`animation = "typewriter"` is rejected on `countdown` — it's only supported on `message`, `gif`, and `image`. For dynamic color, use [`font_color = "rainbow"`](/concepts/color-providers/) or a gradient.
