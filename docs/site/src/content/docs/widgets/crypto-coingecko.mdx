The **crypto.coingecko** widget is provided by the **[crypto](https://github.com/JamesAwesome/led-ticker-plugins/tree/main/plugins/crypto)** package in the led-ticker-plugins monorepo, referenced in config as `type = "crypto.coingecko"`.

- **To add it to your sign:** add `led-ticker-crypto` to `config/requirements-plugins.txt` and restart (`docker compose restart`) — see [Plugins](/plugins/) for the full flow.
- **Full documentation** — options, config examples, and a demo gif — lives in the [crypto package README](https://github.com/JamesAwesome/led-ticker-plugins/blob/main/plugins/crypto/README.md).

<Aside type="tip" title="Want richer crypto layouts?">
  For **major coins** (BTC, ETH, SOL, and the other pairs Twelve Data lists), the
  [**stocks.ticker** widget](/widgets/stocks/) with `provider = "twelvedata"` is the more
  feature-rich option: the same held **card** and full **dashboard** layouts as stocks — hero
  price, sparkline, a pulsing LIVE chip, a scrolling watch column — plus an embeddable,
  trend-colored [`stocks.quote` price token](/widgets/stocks/#embed-a-live-price-anywhere) you
  can drop into any `message`. And it shares **one** free key with your stocks and forex, so a
  mixed rotation (equities + FX + crypto) runs off a single source.

Reach for `crypto.coingecko` here when you want **broad coin coverage** instead: CoinGecko
lists thousands of coins, including the sub-cent long tail (like SHIB) that a majors-only feed
won't carry.

</Aside>

## Features

- **Multi-coin cycling** — displays one ticker card per configured coin, rotating through the list.
- **Three coin-spec styles** — use a single `symbol` + `symbol_id` pair, a `symbol_ids = [...]` list, or a `symbols = [...]` list that auto-resolves each symbol to a unique CoinGecko ID. If a symbol matches more than one coin, the widget raises an error at startup; set `symbol_id` or `symbol_ids` to disambiguate.
- **Optional demo API key** — the keyless free tier is rate-limited to ~5 requests/min. Add `api_key = "..."` in config or set the `COINGECKO_API_KEY` environment variable to use a [CoinGecko demo key](https://www.coingecko.com/en/api/pricing) for a higher rate limit.
- **Adaptive sub-cent formatting** — prices below $0.01 display with enough decimal places to show a meaningful value rather than rounding to zero.

Full options, config examples, and a demo gif live in the [crypto package README](https://github.com/JamesAwesome/led-ticker-plugins/blob/main/plugins/crypto/README.md).
