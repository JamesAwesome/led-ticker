The `etherscan` widget fetches current Ethereum network gas prices from the [Etherscan Gas Oracle](https://etherscan.io/gastracker) and displays three tiers — `Low`, `Avg`, and `High` — in Gwei. Each value is automatically colored by absolute price level: green for cheap gas (≤ 50 Gwei), yellow for moderate (51–70 Gwei), and red for expensive (> 70 Gwei).

:::note[No demo gif — API key required]
The etherscan widget makes live HTTP calls to Etherscan and requires `ETHERSCAN_API_KEY` set in your `.env` file. Neither live network calls nor a real API key are available in CI, so no demo gif is provided. Get a free key at [etherscan.io/apis](https://etherscan.io/apis), then run the widget locally to see it animate.
:::

Add your key to `.env` before running:

```sh
ETHERSCAN_API_KEY=your_key_here
```

## Tips

### `ETHERSCAN_API_KEY` is required

The widget calls `api.etherscan.io` with your key on every fetch. Without a valid key the Etherscan API returns an error object and the widget raises `ValueError` at startup. Get a free key at [etherscan.io/apis](https://etherscan.io/apis) — the free tier allows 5 calls per second and 100,000 calls per day, which is far more than any reasonable polling interval needs.

### Gas coloring is threshold-based, not delta-based

Unlike the `coinbase` and `coingecko` widgets, the color of each gas value reflects its absolute Gwei level (green ≤ 50, yellow ≤ 70, red > 70), not a change vs the previous fetch. A "green" Low tier means gas is cheap right now, not that it went up.

### Gas prices can change rapidly

During network congestion, gas can spike from 20 Gwei to 200 Gwei within minutes. The default 5-minute `update_interval` is appropriate for a passive display; drop to `update_interval = 60` if you need near-real-time visibility during volatile periods.
