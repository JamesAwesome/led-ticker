The `message` widget displays static text. It's the most-used widget — most LED signs are 80% messages with a sprinkling of data widgets.

<DemoGif src="/demos/message-rainbow.gif" caption="message with font_color = 'rainbow'" />

<TomlExample
  title="Minimal example"
  code={`[[playlist.section.widget]]
type = "message"
text = "Hello, world!"`}
/>

## Common patterns

Static text with brand color:

<TomlExample
  code={`[[playlist.section.widget]]
type = "message"
text = "Aerial for Everybody"
font_color = [189, 169, 234]   # brand lavender`}
/>

Inline emoji:

<DemoGif
  src="/demos/message-inline-emoji.gif"
  caption="`:taco:` resolves to the inline pixel-art sprite, with the message text following it"
/>

<TomlExample
  code={`[[playlist.section.widget]]
type = "message"
text = ":taco: Taco Tuesday!"`}
/>

Hi-res font on the bigsign — Inter-Bold at 28px reads well at medium viewing distance:

<TomlExample
  code={`[[playlist.section.widget]]
type = "message"
text = "GRAND OPENING"
font = "Inter-Bold"
font_size = 28
font_color = [225, 48, 108]`}
/>

Typewriter effect with rainbow text:

<DemoGif
  src="/demos/concepts-animations.gif"
  caption="typewriter reveal composed with rainbow font_color — each character types in at its own rainbow hue"
/>

<TomlExample
  code={`[[playlist.section.widget]]
type = "message"
text = "Now Enrolling"
font_color = "rainbow"
animation = "typewriter"`}
/>
