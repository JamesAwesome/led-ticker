The `mlb` widget fetches live game state from MLB's free StatsAPI and renders the tracked team's current series — with distinct display modes for pre-game (opponent and start time), in-game (score, inning, base runners, and balls-strikes-outs in color), post-game ("Final" with win/loss coloring), and postponements. No API key is required.

## Common patterns

### Track two teams in the same section — each widget shows its own team's current series, and they scroll one after the other

<TomlExample
  code={`[[playlist.section.widget]]
type = "mlb"
team = "NYY"
timezone = "America/New_York"

[[playlist.section.widget]]
type = "mlb"
team = "LAD"
timezone = "America/Los_Angeles"`}
/>

### Color the background with your team's brand color and set a faster poll during game hours

<TomlExample
  code={`[[playlist.section.widget]]
type = "mlb"
team = "BOS"
bg_color = [189, 48, 57]
update_interval = 60`}
/>

### Show a west-coast team but format game times in local Pacific time — avoids "Today 10:05 PM" showing up as "Tmrw" on an east-coast display

## Tips

### Poll interval vs. live-game cadence

The `update_interval` you set in TOML controls the background task's outer loop. Internally, the widget already throttles to ~45 s during a live game and ~5 minutes at idle — but only within each outer-loop cycle. Setting `update_interval = 300` (the default) is appropriate for most setups; lowering it to 60 s tightens the cadence without being rude to the public API.

### Offseason behavior

After the World Series ends and before Spring Training begins, no games are scheduled in the ±7-day window the widget queries. The display shows "Season Over" until the schedule populates. For standings and offseason context, pair with the [`mlb_standings`](/widgets/mlb_standings/) widget.

### `final_hold_hours` and late games

The widget shows a completed game's final score for `final_hold_hours` (default 6) after the game's start time. West-coast games that start at 10 PM ET finish around 1–2 AM ET, so the default 6-hour window keeps the score visible through the morning. Raise this if your display runs overnight and you want the score to persist into the next day.
