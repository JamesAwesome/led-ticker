title: mlb widget
description: The MLB scores widget is now the external led-ticker-baseball plugin (type = "baseball.scores").
---

The **mlb** scores widget has moved out of led-ticker core into its own plugin, **[led-ticker-baseball](https://github.com/JamesAwesome/led-ticker-baseball)**, referenced in config as `type = "baseball.scores"`.

- **To add it to your sign:** add `git+https://github.com/JamesAwesome/led-ticker-baseball.git@main` to `config/requirements-plugins.txt` and rebuild — see [Plugins](/plugins/) for the full flow. It uses MLB's free StatsAPI, so no API key is required.
- **Full documentation** — options, display modes (pre-game / in-game / final / postponed), team colors, and screenshots — lives in the [led-ticker-baseball README](https://github.com/JamesAwesome/led-ticker-baseball#readme).

The same plugin also ships the [`baseball.standings`](/widgets/mlb_standings/) widget, the `baseball.roll*` rolling-ball [sprite transition](/transitions/sprite/), and the `:baseball.ball:` emoji. It's another example of led-ticker's [plugin system](/plugins/) — a way to add widgets, transitions, and emoji that aren't part of core.
