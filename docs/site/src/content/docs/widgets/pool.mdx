The **pool** widget is provided by the **[pool](https://github.com/JamesAwesome/led-ticker-plugins/tree/main/plugins/pool)** package in the led-ticker-plugins monorepo, referenced in config as `type = "pool.monitor"`.

- **To add it to your sign:** add `led-ticker-pool` to `config/requirements-plugins.txt` and restart (`docker compose restart`) — see [Plugins](/plugins/) for the full flow. It needs a running InfluxDB v2 server.
- **Full documentation** — options, `ticker` vs `two_row` layouts, InfluxDB setup, and screenshots — lives in the [pool package README](https://github.com/JamesAwesome/led-ticker-plugins/blob/main/plugins/pool/README.md).

Pool is the flagship example of led-ticker's [plugin system](/plugins/) — a way to add widgets that aren't part of core.
