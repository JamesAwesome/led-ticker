The **pool** widget is provided by the **[led-ticker-pool](https://github.com/JamesAwesome/led-ticker-pool)** plugin, referenced in config as `type = "pool.monitor"`.

- **To add it to your sign:** add `git+https://github.com/JamesAwesome/led-ticker-pool.git@main` to `config/requirements-plugins.txt` and rebuild — see [Plugins](/plugins/) for the full flow. It needs a running InfluxDB v2 server.
- **Full documentation** — options, `ticker` vs `two_row` layouts, InfluxDB setup, and screenshots — lives in the [led-ticker-pool README](https://github.com/JamesAwesome/led-ticker-pool#readme).

Pool is the flagship example of led-ticker's [plugin system](/plugins/) — a way to add widgets that aren't part of core.
