The **feeds.rss** widget is provided by the **[led-ticker-feeds](https://github.com/JamesAwesome/led-ticker-feeds)** plugin, referenced in config as `type = "feeds.rss"`. It fetches a remote RSS feed and turns each headline into its own scrolling line in the section.

<DemoGif
  src="/demos-long/widget-rss_feed.gif"
  caption="RSS feed scrolling Nintendo Life headlines, each story in a different color from the default rotation."
/>

- **To add it to your sign:** add `git+https://github.com/JamesAwesome/led-ticker-feeds.git@main` to `config/requirements-plugins.txt` and rebuild — see [Plugins](/plugins/) for the full flow.
- **Full documentation** — options, config examples, and a demo gif — lives in the [led-ticker-feeds README](https://github.com/JamesAwesome/led-ticker-feeds#readme).

See the [plugin system](/plugins/) for how plugin widgets work.
