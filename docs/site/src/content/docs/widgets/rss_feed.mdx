The **rss.feed** widget is provided by the **[rss](https://github.com/JamesAwesome/led-ticker-plugins/tree/main/plugins/rss)** package in the led-ticker-plugins monorepo, referenced in config as `type = "rss.feed"`. It fetches a remote RSS feed and turns each headline into its own scrolling line in the section.

<DemoGif
  src="/demos-long/widget-rss_feed.gif"
  caption="RSS feed scrolling Nintendo Life headlines, each story in a different color from the default rotation."
/>

- **To add it to your sign:** add `git+https://github.com/JamesAwesome/led-ticker-plugins.git@rss-v0.2.0#subdirectory=plugins/rss` to `config/requirements-plugins.txt` and rebuild — see [Plugins](/plugins/) for the full flow.
- **Full documentation** — options, config examples, and a demo gif — lives in the [rss package README](https://github.com/JamesAwesome/led-ticker-plugins/blob/main/plugins/rss/README.md).

See the [plugin system](/plugins/) for how plugin widgets work.
