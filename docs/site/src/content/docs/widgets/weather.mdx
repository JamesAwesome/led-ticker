The **weather.current** widget is provided by the **[weather](https://github.com/JamesAwesome/led-ticker-plugins/tree/main/plugins/weather)** package in the led-ticker-plugins monorepo, referenced in config as `type = "weather.current"`. It shows current conditions and temperature from [WeatherAPI.com](https://www.weatherapi.com/) with a pixel-art condition icon.

<DemoGif
  src="/demos-long/widget-weather.gif"
  caption="Weather showing condition icon, label, and temperature in their real deployed colors."
/>

Install the plugin and set your API key, then add a widget block to your config:

```toml
[[playlist.section]]
mode = "slideshow"

[[playlist.section.widget]]
type = "weather.current"
text = "NYC"
location = "New York"
units = "imperial"
```

You need the plugin installed (see [Plugins](/plugins/)) and `WEATHERAPI_KEY` in your `.env` — free key at [weatherapi.com](https://www.weatherapi.com/).

**Full documentation** — all options, config examples, and a demo gif — lives in the [weather package README](https://github.com/JamesAwesome/led-ticker-plugins/blob/main/plugins/weather/README.md).
