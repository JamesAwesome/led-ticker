The **feeds.weather** widget is provided by the **[led-ticker-feeds](https://github.com/JamesAwesome/led-ticker-feeds)** plugin, referenced in config as `type = "feeds.weather"`. It shows current conditions and temperature from [WeatherAPI.com](https://www.weatherapi.com/) with a pixel-art condition icon.

<DemoGif
  src="/demos-long/widget-weather.gif"
  caption="Weather showing condition icon, label, and temperature in their real deployed colors."
/>

- **To add it to your sign:** add `git+https://github.com/JamesAwesome/led-ticker-feeds.git@main` to `config/requirements-plugins.txt` and rebuild — see [Plugins](/plugins/) for the full flow. You'll also need a free `WEATHERAPI_KEY` set in your `.env`.
- **Full documentation** — options, config examples, and a demo gif — lives in the [led-ticker-feeds README](https://github.com/JamesAwesome/led-ticker-feeds#readme).

See the [plugin system](/plugins/) for how plugin widgets work.
